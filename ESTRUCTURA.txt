SCRIPTS DISPONIBLES
===================

ROOT:
  pnpm dev           Ejecutar API + Web en paralelo
  pnpm build         Compilar todos los packages
  pnpm lint          Ejecutar ESLint
  pnpm lint:fix      ESLint con --fix
  pnpm typecheck     Verificar tipos TypeScript
  pnpm test          Ejecutar tests Vitest

ESPECÍFICOS:
  pnpm --filter @finanzas/api dev        Backend solo
  pnpm --filter @finanzas/web dev        Frontend solo
  pnpm --filter @finanzas/shared build   Compilar shared


VALIDACIONES PRE-COMMIT (Husky + lint-staged)
==============================================
Los commits ejecutan automáticamente:
  1. ESLint --fix
  2. Prettier --write
  3. TypeScript --noEmit

Requisitos para commits exitosos:
  ✓ Sin errores ESLint
  ✓ Código formateado Prettier
  ✓ Sin errores TypeScript
  ✓ Sin uso de `any`


VARIABLES DE ENTORNO (packages/api/.env)
========================================
REQUERIDAS:
  PORT=3001
  NODE_ENV=development
  MONGO_URI=mongodb://...
  REDIS_URL=redis://...
  JWT_SECRET=<32+ caracteres>
  JWT_REFRESH_SECRET=<32+ caracteres>
  ENCRYPTION_KEY=<64+ caracteres hex>

OPCIONALES (APIs de mercado):
  BINANCE_API_KEY, BINANCE_API_SECRET
  CMC_API_KEY
  FINNHUB_API_KEY
